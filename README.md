## Voidblade

N64 Brew Game Jam 2021 entry

Compiled against https://github.com/DragonMinded/libdragon/pull/234 for a little better performance on NTSC

You can build this rom with `libdragon-docker` (https://github.com/anacierdem/libdragon-docker) using the root level Makefile.

Or if your environment supports, start the devcontainer. On Linux, it's capable of building & running your ROM on ares.
